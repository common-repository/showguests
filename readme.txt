== Description ==

Description: Showguests is a plugin for showing the newest quests on your blog. Please visit http://www.tiandiyoyo.com for more information.


== Installation ==

   1. In your FTP, open wp-contents/plugins folder.
   2. Extract plugin zipped file and upload folder named showguests.
   3. In your showguests page, activate showguests.
   4. Insert [sgshow] into your post/page to show it or use widget tools to load it on the slidebar!
   5. The default css is only for the theme 'Freshblog' which using on my blog. So You need do it by yourself when you use other theme(Ex. change fullwidget to widget in my.css).
